# 施工日志 010 — UI 美化 + 广告位预留

**日期**: 2026-05-27
**改动类型**: UI 重构 + 功能预留

**改动内容**:

### UI 美化
1. 配色升级：背景 `#0a0e13`（更深），CSS 变量统一管理（`--bg`, `--surface`, `--accent` 等）
2. 背景加细微网格纹理（24px 间隔）
3. 聊天气泡优化：发送方渐变色（`#1a7f37 → #176b2e`），接收方更深层次，圆角 18px
4. 消息出现动画：`msgIn` translateY + opacity（0.3s）
5. 焚毁动画：CSS class `.burning`（opacity 0 + scale 0.85, 0.4s）
6. 欢迎页卡片式布局：`welcome-card` 带圆角 + 阴影
7. 状态栏：等待连接时指示灯 pulse 动画
8. 邀请码显示：SF Mono 字体 + 深色背景框 + user-select all
9. 输入区压缩、按钮 hover/active 反馈优化
10. 滚动条美化

### 广告位预留（3 处，默认隐藏）
1. **顶部横幅** (`.ad-banner-top`) — 状态栏下方，移动 50px / 桌面 90px 高
2. **底部横幅** (`.ad-banner-bottom`) — 聊天区下方、输入栏上方
3. **插屏广告** (`.ad-interstitial`) — 欢迎页/邀请页/加入页各一张，300×250

广告位通过 `AD_CONFIG` 对象控制：
```javascript
const AD_CONFIG = { bannerTop: false, bannerBottom: false, interstitial: false };
```
设为 `true` 即显示对应占位框。日后替换为 AdSense 等真实代码即可。
